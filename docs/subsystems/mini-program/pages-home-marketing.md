## 首页

### 页面定位

首页是用户打开小程序后的第一个页面，核心目标是**快速引导用户到达关键操作**：购买产品、启动淋浴、查看会员。

### 首页区块定义

| 区块 | 字段 | 说明 |
|---|---|---|
| 门店信息栏 | 当前门店名称 + 切换按钮 | 底部弹出门店选择列表 |
| 快捷功能 | 淋浴控制 / 购买会员 / 人脸管理 | 3 个图标入口 |
| 我的会员卡 | 当前有效会员卡摘要 | 有有效卡时显示；无卡时显示「去购买」引导 |
| 今日优惠 | 限时促销 Banner | 运营配置的活动 Banner |
| 热门产品 | 前 2-3 个产品卡片 | 横向滑动或竖排 |

### 条件展示逻辑

| 条件 | 展示 |
|---|---|
| 用户有有效会员卡 | 会员卡区块显示卡名称 + 到期日期 + 进度条 |
| 用户无有效会员卡 | 会员卡区块显示「暂无会员卡」+ 「去购买」按钮 |
| 用户未录入人脸 | 快捷功能区「人脸管理」图标显示红色角标 `!` |
| 用户在门店内（定位/进入记录） | 「淋浴控制」按钮高亮可点击 |
| 用户不在门店内 | 「淋浴控制」按钮灰色不可点击 |

### 地图模式

| 元素 | 说明 |
|---|---|
| 地图 | 使用微信内置地图组件 `map` |
| 标记点 | 每个门店一个标记，颜色区分状态（绿色=营业中，灰色=非营业） |
| 底部弹窗 | 点击标记弹出门店摘要（名称+距离+状态） |
| 导航 | 点击「导航」调用微信导航或跳转第三方地图 |

| 字段 | 说明 |
|---|---|

## 营销推广

### 活动 Banner

| 配置项 | 说明 |
|---|---|---|
| 展示位置 | 首页「今日优惠」区块 |
| 数据来源 | 管理后台运营配置（[建议值] 第一期由管理后台手动配置，后续可扩展 CMS） |
| 交互 | 点击跳转到产品详情页或活动页 |
| 轮播 | 多个 Banner 自动轮播，间隔 3 秒 |
| 尺寸 | 宽度 100%，高度 120-150px |

### 限时优惠标签

产品卡片上可展示以下标签（由管理后台配置）：

| 标签 | 样式 | 触发条件 |
|---|---|---|
| 限时优惠 | 红色角标 | 产品设置了划线价（originalPrice > price） |
| 热门 | 橙色标签 | 已售数量超过阈值 |
| 推荐 | 蓝色标签 | 手动标记 |
| 新品 | 绿色标签 | 产品创建时间 < 7 天 |

### 分享功能 [建议值]

> **[建议值]** 第一期不开发分享邀请/裂变功能，降低复杂度。后续可根据运营需求扩展。

| 功能 | 第一期 | 后续 |
|---|---|---|
| 分享小程序给好友 | ✅（微信默认能力） | — |
| 分享专属邀请码 | ❌ | 后续 |
| 邀请好友得奖励 | ❌ | 后续 |
| 分享得优惠券 | ❌ | 后续 |

### 动作库字段

| 字段 | 说明 |
|---|---|
| 动作名称 | `name` |
| 分类 | `category`：胸部/背部/腿部/肩部/手臂/核心 |
| 难度 | `difficulty`：初级/中级/高级 |

| 收藏数/浏览数 | 统计值 |
| 是否收藏 | 当前用户是否已收藏 |

> **[建议值]** 动作库内容第一期可**预置 30-50 个基础动作数据**（图片+文字），不自建内容后台。后续如有需要再扩展 CMS 管理能力。

## 接口补充

```
# 门店列表
GET /api/v1/stores
  Query: { latitude?, longitude?, keyword?, status? }
  Response: {
    items: Store[],
    total: number
  }

# 门店详情
GET /api/v1/stores/:id
  Response: Store { id, name, address, phone, businessHours, status,
                    facilities, photos, description, latitude, longitude }

# 动作库
GET /api/v1/exercises
  Query: { category?, keyword?, page, pageSize }
  Response: { items: Exercise[], total: number }

GET /api/v1/exercises/:id
  Response: Exercise { ...全部字段 }

POST /api/v1/exercises/:id/favorite
DELETE /api/v1/exercises/:id/favorite

# 活动 Banner
GET /api/v1/banners?position=home
  Response: { items: Banner[] }
```
