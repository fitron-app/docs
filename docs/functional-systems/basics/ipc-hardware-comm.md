# 工控机与各硬件通信方式

**涉及子系统**：工控机
**核心业务**：定义工控机与各硬件设备之间的通信接口与协议

---

## 通信方式汇总

| 硬件设备 | 通信方式 | 协议/接口 | 说明 |
|---|---|---|---|
| 刷脸机（×2） | 网络 | TCP/IP + HTTP/SDK | 通过局域网与刷脸机通信，获取识别结果 |
| 门锁（AB 门） | GPIO 或 RS485 | 继电器控制 | 低电平触发开锁，门磁反馈状态 |
| 门磁 | GPIO | 数字输入 | 检测门开/关状态 |
| 温湿度传感器 | RS485 | Modbus RTU | 读取温度/湿度数据 |
| 空调控制 | RS485 或 红外 | Modbus / 红外码 | 开关机、模式、温度设定 |
| 灯光控制 | GPIO / RS485 | 继电器 / DMX | 单路或多路灯光开关 |
| 淋浴控制 | GPIO | 继电器控制 | 热水阀开关 |
| 店音播放 | USB / 音频输出 | ALSA / MPD | 播放本地音频文件 |
| 监控摄像头 | 网络 | RTSP / ONVIF | 视频流查看（只读） |
| UPS | USB / RS232 | NUT / 厂商协议 | 电池状态监测 |

---

## RS485 总线规范

- 使用 Modbus RTU 协议
- 波特率：9600 / 19200（按设备规格）
- 地址分配：每类设备按门店预留地址段
