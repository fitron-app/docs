# OTA 更新

**涉及子系统**：工控机、云端 API
**核心业务**：工控机主控程序远程升级，版本管理与回滚

---

## 更新流程

```mermaid
flowchart TD
    Upload["运维人员上传新版本包\n至云端存储"] --> Publish["在管理后台\n发布新版本（指定目标门店）"]
    Publish --> MQTT["云端推送 MQTT 指令\n{type: ota_update, version, url, checksum}"]
    MQTT --> Download["工控机下载更新包\n校验 SHA256"]
    Download --> Install["停止主控程序\n替换程序文件\n重启 systemd service"]
    Install --> Report["工控机上报更新结果\n（成功 / 失败 + 原因）"]
```

---

## 版本管理规则

- 版本号格式：`MAJOR.MINOR.PATCH`（如 `1.2.3`）
- 每个版本包含：程序二进制 + 配置模板 + 变更说明
- 云端保留最近 **10 个**版本包，支持一键回滚到任意历史版本
- 更新前自动备份当前版本程序文件

---

## 安全要求

- 下载链接带时效签名（有效期 1 小时）
- 安装前校验 SHA256 完整性
- 更新包通过 HTTPS 传输
