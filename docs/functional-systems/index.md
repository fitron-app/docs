# 功能系统概览

功能系统以**业务能力**为边界划分，横跨多个子系统共同实现一项完整的业务功能。同一个功能系统的代码可能同时存在于工控机、云端 API、小程序等多个子系统中。

## 功能系统清单

| 功能系统 | 核心业务 | 涉及子系统 |
|---|---|---|
| [刷脸系统（AB 门）](./face-recognition) | 人脸识别进出控制，AB 门状态机 | 工控机、云端 API、小程序 |
| [门禁系统](./access-control) | 门锁物理控制、状态监控、远程操作 | 工控机、云端 API、管理后台 |
| [产品/计费系统](./product-system) | 套餐定价、会员有效期、次卡规则 | 云端 API、管理后台、小程序 |
| [订单系统](./order-system) | 购买、支付、核销、退款全流程 | 云端 API、管理后台、小程序 |
| [用户系统](./user-system) | 注册、人脸绑定、会员状态管理 | 云端 API、管理后台、小程序 |
| [优惠券系统](./coupon-system) | 内部优惠券类型、发放、核销规则 | 云端 API、管理后台、小程序 |
| [外部平台券码核销](./external-voucher) | 抖音/美团等外部渠道券码兑换，自动创建订单 | 云端 API、管理后台、小程序 |
| [淋浴系统](./shower-system) | 淋浴启停控制、使用时长管理 | 工控机、云端 API、小程序 |
| [硬件控制系统](./hardware-control) | 灯光、设备远程控制、状态监控 | 工控机、云端 API、管理后台 |
| [数据分析系统](./analytics) | 经营报表、进出统计、飞书集成 | 云端 API、管理后台 |
| [管理后台门店上下文](./admin-store-context) | 全体店/单店模式切换，自动附加门店筛选范围 | 管理后台、云端 API |
| [紧急安全系统](./emergency-safety) | 击碎玻璃触发声光报警、强制全开所有门、上报总部 | 工控机、云端 API、管理后台 |
