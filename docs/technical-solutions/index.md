## 背景说明

健身房工控机（RK3576）部署在全国各门店，运维条件受限：

- 门店数量：初期百店，目标千店级别
- 无专业运维人员驻场
- 网络环境参差不齐（一线城市光纤 / 小城市宽带 / 4G 备用）
- 出现故障时人工到场成本极高

因此，工控机侧的**自愈能力**和**远程可达性**是系统稳定运行的核心保障，这三个技术方案共同构成工控机的运维底座。
