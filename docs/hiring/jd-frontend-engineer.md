# 前端工程师

**薪资**：13,000 ~ 15,000 元 / 月  
**工作年限**：0 ~ 3 年  
**学历要求**：本科及以上
**专业要求**：计算机科学、软件工程等计算机相关专业

---

## 关于我们

我们是一家专注于智能健身场景的科技公司，正在打造新一代**无人值守连锁健身房**系统。产品覆盖用户端小程序、运营管理后台、IoT 硬件控制等多个方向，全程基于 AI 辅助开发，工程文化现代高效。

---

## 岗位职责

1. 负责 **公司官网** 、 **微信小程序** 与 **SaaS 服务后台** 等业务系统的开发与迭代
2. 与后端系统共同进行接口设计、调试
3. 与产品、测试等团队协作，参与项目全生命周期管理，确保项目按时交付;
4. 参与页面性能优化与用户体验改善
5. 参与代码评审，维护前端工程质量

---

## 任职要求

1. 计算机相关专业本科及以上学历
2. **必须熟练使用 AI 编程工具** 进行日常开发
3. 熟练掌握 **React** 或 **Vue.js**（任一框架扎实即可），有 TypeScript 使用经验
4. 有 **微信小程序** 开发经验者优先，熟悉小程序基础 API 与授权流程
5. 熟悉前后端 API 对接，能够独立完成模块开发与自测
6. 具备良好的代码习惯，了解浏览器、小程序的相关机制，有一定的 UI 审美，能落地设计稿
7. 沟通能力强，能在团队中灵活协作，主动推进任务

---

## 加分项

- 有中后台系统（数据看板、表格、图表）开发经验
- 了解数据可视化（ECharts 或类似图表库）
- 有 uni-app 或跨端开发经验
- 有过完整项目从零到上线的经历
