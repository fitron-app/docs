### 网络设备

| 序号 | 物料名称 | 规格/要求 | 数量 | 状态 | 备注 |
|---|---|---|---|---|---|
| 21 | 工业级双路 WAN 路由器 | 同时支持宽带（WAN 口）+ 4G IoT SIM 卡；宽带为主路，4G 为自动热备；支持 API 或 MQTT/webhook 上报当前在线 WAN 类型；千兆有线口 ≥ 2 | 1 | 🔄 | 工控机通过有线连接此路由器；宽带断开时自动切换 4G 并触发工控机报警逻辑 |

> **网络监测逻辑（软件侧）**：工控机通过路由器 API 或主动探测（ping 宽带网关）实时感知当前 WAN 类型。若检测到由宽带切换至 4G（即宽带断开），工控机立即通过 4G 网络向云端服务器发送告警事件，管理后台展示告警并推送通知。恢复宽带后发送恢复通知。
