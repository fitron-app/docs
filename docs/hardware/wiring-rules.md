# 内部接线规则

本文档规定控制箱内部及箱体到外部设备的接线规范，确保安装、维护、排障的一致性。

> **安全警告**：所有强电（220VAC）接线操作必须由持证电工执行，操作前断开总电源。

---

## 接线规范总则

1. **颜色规范**：相线（火线）红色，零线蓝色，地线黄绿色，DC 正极红色，DC 负极黑色
2. **线径规范**：220VAC 主回路 ≥ 2.5mm²，控制信号线 0.5-1.0mm²，GPIO 信号线 0.3mm²
3. **标签规范**：每根导线两端必须套管号管，格式：`[设备代码]-[信号名称]`，如 `DOOR_A-LOCK+`
4. **线槽规范**：强电与弱电走线槽分离，间距 ≥ 10cm
5. **接地规范**：控制箱金属壳体、工控机、路由器等设备外壳均须可靠接地

---

## 设备代码规范

| 代码 | 设备 |
|---|---|
| `IPC` | 工控机（Industrial PC） |
| `PSU` | 开关电源（Power Supply Unit） |
| `UPS` | UPS 不间断电源 |
| `RLY_DOOR` | 门锁继电器模块 |
| `RLY_LIGHT` | 灯控继电器模块 |
| `LOCK_A` | A 门电磁锁 |
| `LOCK_B` | B 门电磁锁 |
| `MAG_A` | A 门门磁传感器 |
| `MAG_B` | B 门门磁传感器 |
| `PIR` | 隔离间人体传感器 |
| `BTN_IN` | 入口开门按钮 |
| `BTN_OUT` | 隔离间出门按钮 |
| `CAM` | 人脸识别摄像头 |
| `RTR` | 路由器/交换机 |

## 强电接线（220VAC）

```
市电入线（220VAC）
      │
  [断路器 2P 16A]
      │
  [UPS 输入]
      │
  [UPS 输出]
      ├──────────────────► 工控机（IPC）电源
      ├──────────────────► 开关电源（PSU，输出 DC 12V）
      ├──────────────────► 路由器/交换机
      └──────────────────► 灯控继电器输出公共端（如控制 220V 灯）
```

**强电接线要点：**
- UPS 在断路器之后，保证断电时全部设备切换至电池供电
- 工控机、路由器均通过 UPS 供电，保证断市电后仍能正常运行
- 220VAC 灯控接线需使用带熔断保护的接线端子

---

## 弱电接线（DC 12V）

```
[PSU 输出 DC12V+]───[接线端子排 DC12V+汇流]
        │
        ├──► LOCK_A 正极（门锁线圈 +）
        ├──► LOCK_B 正极
        ├──► MAG_A 供电 +
        ├──► MAG_B 供电 +
        ├──► PIR 供电 +（VCC）
        ├──► BTN_IN 供电 +（如有源按钮）
        └──► RLY_DOOR 线圈供电 +

[PSU 输出 DC12V-]───[接线端子排 DC12V-/GND 汇流]
        │
        └──► 以上所有设备的负极/GND
```

## GPIO 接线（工控机 ↔ 外部设备）

> 工控机 GPIO 电平以实际选型为准（通常 3.3V 或 5V TTL）。
> 若 GPIO 无法直接驱动继电器线圈，需经过 ULN2003 或光耦隔离驱动板。

### 输出信号（IPC → 设备）

| 信号名称 | 管脚（待填） | 目标设备 | 说明 |
|---|---|---|---|
| `DOOR_A_CTRL` | GPIO_XX | `RLY_DOOR` 通道 1 输入 | 高电平 → 继电器闭合 → 电磁锁通电（解锁或锁定，视锁型） |
| `DOOR_B_CTRL` | GPIO_XX | `RLY_DOOR` 通道 2 输入 | 同上，控制 B 门 |
| `LIGHT_ZONE1` | GPIO_XX | `RLY_LIGHT` 通道 1 输入 | 健身区主灯控制 |
| `LIGHT_ZONE2` | GPIO_XX | `RLY_LIGHT` 通道 2 输入 | 隔离间灯控制 |
| `LIGHT_ZONE3` | GPIO_XX | `RLY_LIGHT` 通道 3 输入 | 其他分区（待定） |

### 输入信号（设备 → IPC）

| 信号名称 | 管脚（待填） | 来源设备 | 说明 |
|---|---|---|---|
| `MAG_A_STATE` | GPIO_XX | `MAG_A` 信号输出 | 低电平 = 门关闭（常闭型门磁） |
| `MAG_B_STATE` | GPIO_XX | `MAG_B` 信号输出 | 同上 |
| `PIR_STATE` | GPIO_XX | `PIR` 信号输出 | 高电平 = 有人检测到 |
| `BTN_IN_TRIG` | GPIO_XX | `BTN_IN` 信号输出 | 低电平触发（按下 = 拉低） |
| `BTN_OUT_TRIG` | GPIO_XX | `BTN_OUT` 信号输出 | 同上 |

## 门锁继电器接线详图

```
IPC GPIO ──► RLY_DOOR 通道1 控制输入

RLY_DOOR 通道1 触点：
  COM（公共端）──► PSU DC12V+
  NO（常开端）──► LOCK_A 线圈 +
  
LOCK_A 线圈 - ──► PSU DC12V-（GND）

（B 门同理）
```

> 注意：若电磁锁为**常开型**（断电开门），接线逻辑不变，但锁的物理状态与通断电关系相反，需在软件层面处理好逻辑。

---

## 门磁传感器接线

```
MAG_A（常闭型，两线）：
  线1 ──► PSU DC12V-（GND）
  线2 ──► IPC GPIO 输入引脚 + 上拉电阻（4.7kΩ 接 VCC）

门关闭：接点闭合，GPIO 读低电平
门打开：接点断开，GPIO 读高电平（上拉）
```

---

## 人体传感器接线

```
PIR 传感器（三线：VCC / GND / OUT）：
  VCC ──► PSU DC12V+（或 5V，视传感器型号）
  GND ──► PSU DC12V-
  OUT ──► IPC GPIO 输入引脚
  
有人：OUT 高电平
无人：OUT 低电平
```

---

## 摄像头接线

| 摄像头类型 | 接线方式 |
|---|---|
| USB 摄像头 | 直接连接工控机 USB 接口，无需外部电源 |
| 网络摄像头（IP Camera） | 网线接交换机，工控机通过 RTSP 拉流 |

---

## UPS 通信接线

```
UPS RS232/USB 接口 ──► 工控机串口/USB
（用于读取 UPS 状态：电量、市电状态等）
```

> 不同 UPS 品牌通信协议不同，需参照具体型号文档配置工控机端驱动。

---

## 接线完成检查清单

- [ ] 所有线缆两端套好管号管，标签清晰
- [ ] 强电/弱电线槽分离
- [ ] 所有设备外壳接地
- [ ] 断路器、UPS 供电路径正确
- [ ] 继电器触点额定电流满足电磁锁要求
- [ ] GPIO 信号电平与工控机匹配（必要时加电平转换/光耦隔离）
- [ ] 门磁上拉电阻已安装
- [ ] 通电前用万用表检查无短路
- [ ] 通电后逐路验证信号正确性（GPIO 高低电平与实际门锁/传感器状态一致）
